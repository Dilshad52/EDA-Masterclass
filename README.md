### Retail Exploratory Data Analysis

## Project Objective:
Perform EDA on a cleaned retail dataset to identify trends, patterns and business opportunities.

## Tools Used:
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed:
- Dataset Overview
- Summary Statistics
- Revenue Distribution
- Revenue by Customer
- Revenue vs Profit Analysis
- Correlation Analysis

## Key Findings:
- Revenue is concentrated among a small number of customers.
- Revenue and profit have a strong positive relationship.
- One customer contributes significantly more than others.
